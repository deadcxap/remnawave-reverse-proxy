## Установка скрипта

:::danger[Важно]
Скрипт нужно запускать с **правами root**.
:::

Введите данную команду на вашем сервере:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

:::note
Если вы хотите установить версию 1.6.16, вы можете использовать следующую команду:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/tags/v.1.7.5/install_remnawave.sh)
```
:::

## Добавить ноду в панель

<Steps>

1. Выберите пункт `Установка компонентов Remnawave`

    ![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/main_menu.png)

2. Затем выберите пункт `Добавить ноду в панель`

    ![Add node to panel RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/add_node_to_panel.png)

3. Введите ваш новый домен `selfsteal` который будет использоваться для подключения ноды к панели, а так же название для самой ноды.

    ![Add node to panel2 RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/add_node_to_panel2.png)

4. После успешного добавления ноды в панель, запустите скрипт на вашем новом сервере, выберите пункт `Установка компонентов Remnawave`, а затем [`Установить только ноду`](https://wiki.egam.es/ru/installation/node-only/).

</Steps>

## Обновление скрипта
