## Установка скрипта

:::danger[Важно]
Скрипт нужно запускать с **правами root**.
:::

Введите данную команду на вашем сервере:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Установка панели и ноды
:::danger[Важно]
Этот способ установки не рекомендуется для использования в производственных средах, поскольку он отсутствует в официальной документации Remnawave.

Для оптимального результата и стабильной работы, пожалуйста, выбирайте методы установки [`Только панель`](https://wiki.egam.es/ru/installation/panel-only/) или [`Только нода`](https://wiki.egam.es/ru/installation/node-only/) в зависимости от ваших потребностей.
:::

:::note
Если вы хотите установить версию 1.6.16, вы можете использовать следующую команду:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/tags/v.1.7.5/install_remnawave.sh)
```
:::

<Steps>

1. Выберите пункт `Установка компонентов Remnawave`

    ![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/main_menu.png)

2. Затем выберите пункт `Установить панель и ноду на один сервер`

    ![Install panel and node RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/install_panel_node.png)

3. После завершения процесса скрипт автоматически перезапустит панель и предоставит все необходимые данные для входа в неё.

    ![Menu end RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/menu_end.png)

</Steps>

:::danger[Важно]
После завершения установки скрипт отобразит уникальный URL-адрес для входа в панель управления.  
**Обязательно сохраните этот URL-адрес.** Он содержит секретный ключ, необходимый для доступа.

Пример ссылки для входа:
```
https://p.example.com/auth/login?SECRET_KEY=SECRET_KEY
```
Без этого URL-адреса **вы не сможете** войти в панель.
:::
